# Sabo Cat Market Wizard
Real-time crypto market trend dashboard powered by AI and Solana.

## Setup
1. Clone: `git clone https://github.com/0xgatoexe/sabo-cat.git`
2. Install: `npm install express ws`
3. Run: `node server.js`
4. Visit: `http://localhost:3000`

## Features
- Real-time Solana AI and crypto asset trend charts.
- Matrix-style background with Sabo Cat imagery.
